Para importar un proyecto
1) Ingresar el siguiente comando -> cd [ruta de la carpeta donde pondras el proyecto]
2) Pegar la ruta de clonacion del proyecto -> git clone https://github.com/pxdalza/VSFundamentals.git
3) esperar que se descargue y usar la aplicacion
